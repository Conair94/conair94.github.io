# Project Context: Personal Website for Connor Lockhart

## Overview
This is a Jekyll-based personal website for Connor Lockhart, a Math PhD student at the University of Maryland. 
The website serves as a portfolio for job applications, specifically targeting the **Quantitative Finance** and **Machine Learning** industries.

## Goals
1.  **Showcase Projects:** Prominently display GitHub projects, especially those related to Quant Finance and ML.
2.  **Academic Work:** Display publications, talks, and teaching history.
3.  **CV/Resume:** Provide easy access to the CV.
4.  **Professional Image:** Clean up the site to look professional and ready for recruiters.

## Key Focus Areas
-   **Quantitative Finance**
-   **Machine Learning**
-   **Mathematics (Model Theory, Probability, Geometry, Combinatorics)**

## Technical Details
-   **Static Site Generator:** Jekyll
-   **Theme:** Minimal Mistakes (or similar derivative)
-   **Hosting:** GitHub Pages
-   **Configuration:** `_config.yml`
-   **Data:** `_data/navigation.yml`, `_data/authors.yml`
-   **Content:** `_pages/`, `_posts/`, `_portfolio/`, `_publications/`, `_talks/`, `_teaching/`
